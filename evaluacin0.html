Evaluación

Pregunta Verdadero-Falso

Pregunta 1

Un procesador de texto, en esencia, es el equivalente digital de una calculadora antigua.

Retroalimentación

Falso

Según el video, un procesador de texto es el equivalente digital de las máquinas de escribir antiguas.

Pregunta 2

El Área de Trabajo es el único lugar donde se pueden insertar imágenes y dibujos, además del texto.

Retroalimentación

Verdadero

El Área de Trabajo es el espacio principal y versátil donde se inserta todo el contenido del documento, incluyendo texto y elementos gráficos.

Pregunta 3

Microsoft Word es un ejemplo de editor de texto desarrollado bajo un sistema de código abierto (gratuito).

Retroalimentación

Falso

Microsoft Word es un software de licencia de pago. El video menciona a Open Office como ejemplo de editor de código abierto (gratuito).

Pregunta 4

Las Barras de Herramientas y la Cinta de Opciones son las que permiten moverte en el documento en forma vertical u horizontal.

Retroalimentación

Falso

Esa función le corresponde a las Barras de Desplazamiento. La Barra de Herramientas/Cinta permite acceder a los comandos.
